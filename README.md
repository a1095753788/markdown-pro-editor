# Markdown Pro Editor

江苏省沭阳县寄意星电子商务经营部制作

一个功能强大的 Markdown 编辑器，专为 Windows 系统设计。支持实时预览、数学公式、代码高亮、自定义打印和格式编辑。

## 功能特性

### 核心功能

- ✅ **实时预览**：左右分割布局，编辑时即时看到预览效果
- ✅ **数学公式支持**：完整的 LaTeX 数学公式支持，使用 KaTeX 引擎
- ✅ **代码高亮**：支持 100+ 种编程语言的语法高亮
- ✅ **文件操作**：新建、打开、保存 Markdown 文件
- ✅ **HTML 导出**：将文档导出为独立的 HTML 文件

### 打印功能

- 📄 **自定义纸张尺寸**：支持 A4、A3、Letter、Legal
- 📐 **灵活的方向设置**：纵向或横向打印
- 📏 **精确的边距控制**：自定义上下左右边距（毫米）
- 🔤 **字体大小调整**：8-24pt 可调
- 📊 **行高设置**：1-3 倍可调

### 格式编辑

- 🎨 **字体选择**：无衬线、衬线、等宽三种字体族
- 📝 **排版控制**：标题大小、正文大小、行高、字间距
- 🌈 **颜色自定义**：文本颜色和背景颜色可调
- 👁️ **实时预览**：格式编辑时实时预览效果

### 编辑工具

- **快速格式化**：工具栏按钮快速插入常用 Markdown 元素
- **标题快捷**：快速插入 H1、H2 标题
- **列表支持**：快速创建有序和无序列表
- **代码块**：快速插入代码块
- **链接和图片**：便捷的链接和图片插入
- **引用块**：快速插入引用

## 技术栈

- **前端框架**：React 19
- **样式**：Tailwind CSS 4
- **Markdown 渲染**：markdown-it
- **数学公式**：KaTeX
- **代码高亮**：highlight.js
- **UI 组件**：shadcn/ui
- **构建工具**：Vite

## 使用指南

### 基本操作

1. **新建文档**：点击"新建"按钮或按 Ctrl+N
2. **打开文件**：点击"打开"按钮或按 Ctrl+O，选择 .md 文件
3. **编辑内容**：在左侧编辑区输入 Markdown 内容
4. **实时预览**：右侧预览区实时显示渲染结果
5. **保存文件**：点击"保存"按钮或按 Ctrl+S 下载文件

### 格式化文本

使用工具栏按钮快速格式化：

- **加粗**：选中文本，点击加粗按钮或按 Ctrl+B
- **斜体**：选中文本，点击斜体按钮或按 Ctrl+I
- **代码**：选中文本，点击代码按钮或按 Ctrl+`
- **标题**：点击标题按钮自动插入标题符号
- **列表**：点击列表按钮自动插入列表符号

### 插入数学公式

#### 行内公式

使用单个 `$` 包围公式：

```markdown
这是行内公式：$E = mc^2$
```

#### 块级公式

使用双 `$$` 包围公式：

```markdown
$$\int_0^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}$$
```

### 打印文档

1. 点击"打印"按钮打开打印设置对话框
2. 选择纸张尺寸（A4、A3、Letter、Legal）
3. 选择方向（纵向或横向）
4. 调整边距（毫米）
5. 设置字体大小和行高
6. 点击"打印"按钮进行打印

### 导出为 HTML

1. 点击"导出 HTML"按钮
2. 浏览器会自动下载 HTML 文件
3. 可以在任何浏览器中打开 HTML 文件

### 自定义格式

1. 点击工具栏中的"格式"按钮
2. 在右侧面板中调整：
   - 字体族（无衬线、衬线、等宽）
   - 标题大小（20-48px）
   - 正文大小（12-20px）
   - 行高（1-2.5）
   - 字间距（-2 到 4px）
   - 文本颜色
   - 背景颜色
3. 预览区会实时显示效果

## Markdown 语法支持

编辑器支持标准 Markdown 语法：

- **标题**：# H1, ## H2, ### H3, 等
- **文本格式**：**粗体**、*斜体*、***粗斜体***、~~删除线~~
- **列表**：无序列表 (-)、有序列表 (1.)、嵌套列表
- **代码**：行内代码 (\`)、代码块 (```)
- **引用**：> 引用块
- **链接**：[文本](url)
- **图片**：![alt](url)
- **表格**：Markdown 表格语法
- **水平线**：---、***、___
