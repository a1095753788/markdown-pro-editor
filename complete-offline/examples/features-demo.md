# Markdown Pro Editor - 功能演示

欢迎使用 **Markdown Pro Editor**！这是一个功能强大的 Markdown 编辑器，专为 Windows 系统设计。

## 核心功能

### 1. 实时预览

编辑器左侧是编辑区，右侧是实时预览区。当您输入 Markdown 时，预览会立即更新。

### 2. 数学公式支持

编辑器完全支持 LaTeX 数学公式，包括：

- **行内公式**：$E = mc^2$
- **块级公式**：
  $$\int_a^b f(x) dx$$

### 3. 代码高亮

编辑器支持多种编程语言的代码高亮：

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```

```javascript
function factorial(n) {
  if (n === 0 || n === 1) return 1;
  return n * factorial(n - 1);
}
```

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

## 编辑功能

### 文件操作

- **新建**：创建新的 Markdown 文档
- **打开**：打开本地 Markdown 文件
- **保存**：下载当前文档为 Markdown 文件
- **导出 HTML**：导出为 HTML 文件

### 格式工具

编辑器提供快速格式化工具：

- **加粗** (Ctrl+B)
- **斜体** (Ctrl+I)
- **代码** (Ctrl+`)
- **标题** (Ctrl+1, Ctrl+2)
- **列表** (Ctrl+L)
- **链接** (Ctrl+K)
- **图片** (Ctrl+Shift+I)
- **引用** (Ctrl+Q)

### 打印功能

编辑器支持自定义打印设置：

- **纸张尺寸**：A4、A3、Letter、Legal
- **方向**：纵向或横向
- **边距**：自定义上下左右边距
- **字体大小**：调整打印时的字体大小
- **行高**：调整打印时的行高

### 格式编辑

通过格式编辑面板，您可以自定义文档的外观：

- **字体族**：无衬线、衬线、等宽
- **标题大小**：20-48px
- **正文大小**：12-20px
- **行高**：1-2.5
- **字间距**：-2 到 4px
- **文本颜色**：自定义颜色
- **背景颜色**：自定义背景
